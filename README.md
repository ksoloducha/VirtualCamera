# VirtualCamera

## Keys

#### Move

* up - move forward
* down - move back
* right - move right
* left - move left
* W - move up
* S - move down

#### Rotate
* Z - rotate OX left
* X - rotate OX right
* C - rotate OY left
* V - rotate OY right
* B - rotate OZ left
* N - rotate OZ right

#### Other actions

* Q - zoom in
* A - zoom out
* P - switch display option
